ParseRule Class

Single named parsing grammar rule.

public sealed class ParseRule : GrammarConstruct

⊟ Remarks

Each ParseRule should only be added to one and only one Parser. It is an error to add a given ParseRule to more than one Parser.

ParseRules will most often be generated from a grammar specification using the ParserCompiler rather than being created manually.
