Parser<T> Generic Class

Parser instance which produces tokens based on expression grammar rules.

public sealed class Parser<T>

⊟ Type Parameters

T
Type of the return value. This is the type which the parser returns after a successful parsing. Parsers which convert the result to a particular type can be created using ConvertingResultUsing().

⊟ Remarks

The first grammar rule specified is always the root grammar rule (also known as the root or starting production), which represents the overall entity being parsed.

Instances of this class may be safely used for concurrent parsing by multiple threads, as long as the parser and its rules are not modified during parsing.

⊟ MaxErrors Property

The maximum number of errors to report while parsing.

public int MaxErrors { set; get; }

⊟ Value

Number of errors to tolerate before a SyntaxException is thrown.

⊟ Remarks

The default is 1 (parsing fails on the first error).

Setting MaxErrors to a higher number will not prevent a SyntaxException from being thrown; rather, it will cause the parser to make an attempt to bypass syntax errors while recording them. The SyntaxException will then contain details of the errors encountered.

⊟ Parse Method

Parses the specified text and returns the result.

public T Parse (string text, params object[] args)

⊟ Parameters

text
Text to parse.
args
Optional arguments for token conversion.

⊟ Returns

Parsed and converted result.

⊟ Exceptions

Type Reason
Naucera.Iambic.SyntaxException Thrown if parsing fails on syntax errors which were unrecoverable (or MaxErrors was reached).

⊟ Remarks

Each rule generates a token, with the components of the rule as the token's children, and each token is tagged with a value if a corresponding tag has been registered using Tagging().

The final result is returned by applying a conversion specified upon construction (or by using ConvertingResultUsing()) to the token produced by the root grammar rule.
