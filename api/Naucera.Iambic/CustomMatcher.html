Matches the given text by searching beginning at the specified offset, returning the number of characters matched from the position indicated by matchOffset. If no match is found, -1 is returned instead.

Subclasses may optionally implement this method to support error recovery. If error recovery is not required (the default, when the MaxErrors property of a Parser is set to 1), this method need not be implemented.

The default implemention throws NotImplementedException.
