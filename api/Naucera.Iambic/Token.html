Token Struct

A parsed token representing a node/subtree of a parse tree.

public struct Token

⊟ Remarks

Tokens may contain nested children. Tokens may also be tagged with a value (accessible via the Tag property).

⊟ ChildTags Property

A sequence of values that the children of this token have been tagged with.

public IEnumerable<object> ChildTags { get; }

⊟ Value

Sequence of tag values for this token's children.

⊟ Remarks

Values are returned in the same sequence as the children they are from.

If no tag has been defined for a child, or tagging has not yet been performed, then the tag value will be null in the sequence for that child.
